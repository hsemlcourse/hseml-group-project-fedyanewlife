## Описание задачи

**Задача:** Предсказание общего дохода игры (задача регрессии).

**Датасет:** Android Games EDA Ready Dataset (взято с сайта [Kaggle](https://www.kaggle.com/datasets/tsmgofficial/android-games-eda-ready-dataset)).
Это синтетический датасет, специально созданный для изучения ML. В нём есть признаки разных типов и пропущенные значения.

**Целевая метрика:** RMSE (основная), дополненная MAE и R².

RMSE выбрана основной метрикой, поскольку она чувствительна к большим ошибкам. Мы прогнозируем доход, поэтому сильные отклонения нежелательны.

MAE дополнит RMSE и покажет абсолютную величину ошибки без квадратичного преувеличения. Таким образом, можно будет делать логический вывод: "Модель ошибается на X долларов".

Наконец, R² позволит оценить, насколько хорошо модель описывает данные в целом. Мы сможем сравнить модели в единой шкале.

## Запуск

Создайте файл `.env` в корне проекта и заполните его по примеру ниже:
```dotenv
JUPYTER_TOKEN=<токен-пароль для jupyter>
````

Запуск проекта осуществляется командой `make up` с помощью Docker.

Полный список команд представлен в таблице ниже:

| Команда | Описание |
|---|---|
| `make lint` | Проверка кода линтером flake8 |
| `make up` | Запуск проекта в Docker |
| `make down` | Остановка контейнеров |
